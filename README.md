## 🔹 Task 1 – Iris Flower Classification

**Objective:**  
Developed a machine learning classification model to predict Iris flower species using flower measurements.

**Highlights**
- Exploratory Data Analysis (EDA)
- Data Visualization
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Model Evaluation

**Tools & Technologies**
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
