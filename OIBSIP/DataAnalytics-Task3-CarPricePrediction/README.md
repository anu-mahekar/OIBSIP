# Car Price Prediction

## 📌 Objective
Predict the selling price of used cars using Machine Learning regression models.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Models Used
- Linear Regression
- Random Forest Regressor

## 📈 Evaluation Metrics
- MAE
- RMSE
- R² Score

## 🚀 Outcome
Compared regression models and selected the best-performing model based on evaluation metrics.
