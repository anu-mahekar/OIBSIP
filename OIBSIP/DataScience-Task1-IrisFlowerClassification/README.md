# Iris Flower Classification

## 📌 Objective
Build a Machine Learning model to classify Iris flowers into Setosa, Versicolor, and Virginica based on flower measurements.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## 🚀 Outcome
Compared multiple classification models and selected the best-performing model based on accuracy and evaluation metrics.
