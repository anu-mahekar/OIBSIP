# Email Spam Detection

## 📌 Objective
Build a Natural Language Processing (NLP) model to classify emails as Spam or Ham.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- TF-IDF
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Models Used
- Multinomial Naive Bayes
- Logistic Regression

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## 🚀 Outcome
Developed an NLP-based email spam classifier capable of distinguishing spam from legitimate emails with high accuracy.
